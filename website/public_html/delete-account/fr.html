Demande de Suppression de Compte

Si tu souhaites supprimer ton compte Nuveli et toutes les données associées, il existe deux méthodes :

1. Depuis l'application (Recommandé)

Le moyen le plus rapide :

  • Ouvre l'application Nuveli
  • Va dans Paramètres → Supprimer le compte
  • Sur l'écran de confirmation, appuie sur "Supprimer définitivement mon compte"

Ton compte et toutes tes données seront définitivement supprimés immédiatement.

2. Par demande email

Si tu ne peux pas accéder à l'application, envoie-nous un email avec les informations suivantes :

  • Ton adresse email enregistrée
  • Objet : "Demande de suppression de compte"
  • Envoie depuis la même adresse email utilisée pour créer le compte (pour vérification)
📧 Envoyer une demande email

Les demandes sont traitées dans les 3 jours ouvrables et tu recevras un email de confirmation.

Données qui seront supprimées

Lorsque tu supprimes ton compte, toutes les données suivantes seront définitivement supprimées :

  • Tes informations de compte (email, profil)
  • Tous les enregistrements et analyses de repas
  • Données de suivi de l'eau et du poids
  • Historique de chat avec le coach IA
  • Tokens de notification de l'appareil
  • Statistiques d'utilisation

⚠️ Action irréversible: Une fois ton compte supprimé, les données ne peuvent pas être récupérées. Cette action ne peut pas être annulée.

Période de Conservation

Après la suppression du compte :

  • Données analytiques anonymes : Complètement supprimées dans les 90 jours (conforme RGPD/KVKK)
  • Obligations légales : Certaines données peuvent être conservées pour des raisons fiscales/légales (max 5 ans) sous forme anonyme
  • Systèmes de sauvegarde : Supprimées des sauvegardes dans les 30 jours
