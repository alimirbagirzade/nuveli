# MVP Kapsam

Bu dosya, Nuveli MVP'sine giren ve girmeyen özellikleri tanımlar.
V2/V3 olarak işaretlenen hiçbir özellik MVP koduna dahil edilemez.

---

## MVP İçinde

### Onboarding & Kabul
- [ ] Yaş kapısı ve wellness scope onayı (5 ekran)
- [ ] Hedef seçimi, profil adımları
- [ ] Koç persona seçimi
- [ ] Bildirim tercihi

### Meal Takibi
- [ ] Fotoğrafla meal analizi (OpenAI Vision)
- [ ] Manuel meal girişi (fallback)
- [ ] Analiz sonucu onaylama / düzenleme
- [ ] Günlük meal listesi ve silme

### Ana Ekran
- [ ] Günlük kalori özeti
- [ ] Hızlı aksiyon butonları (meal ekle, su, kilo, check-in)
- [ ] Koç kartı
- [ ] Craving prompt kartı
- [ ] Mini ilerleme grafiği

### Günlük Yaşam Döngüsü
- [ ] Su takibi (hızlı ekle)
- [ ] Kilo girişi
- [ ] Günlük check-in (ruh hali)

### AI Koç
- [ ] Yazılı koç yanıtı
- [ ] Kısa sesli koç yanıtı (TTS)
- [ ] Boş gün & kurtarma günü akışı

### İlerleme
- [ ] Haftalık özet ekranı
- [ ] Temel aylık içgörü (3 kural)

### Premium
- [ ] RevenueCat entegrasyonu
- [ ] Paywall ekranı
- [ ] Trial deneyimi (7 gün)
- [ ] Feature gating (backend'den)

### Bildirim & Destek
- [ ] Push token kaydı
- [ ] Bildirim tercihleri
- [ ] Support ekranı
- [ ] "AI nasıl çalışır" ekranı
- [ ] Hesap silme

---

## MVP Dışı (V2 / V3)

- Barkod taraması
- Restoran menüsü entegrasyonu
- Beslenme hedefi kişiselleştirme (makro split)
- Sosyal / arkadaş özelliği
- Wearable entegrasyonu
- Öğün planlama
- Alışveriş listesi
- Koç ile video/görüntülü görüşme
- Çoklu dil desteği (v1 sadece Türkçe)
- Web uygulaması

---

## MVP Çıkış Kriteri

1. Onboarding → Home → Meal log → Coach döngüsü uçtan uca çalışır.
2. Free ve premium akışı test edilmiş olur.
3. Delete account görünür ve çalışır.
4. App Store & Google Play submission kurallarına uygun.
