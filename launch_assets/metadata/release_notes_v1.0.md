## 🚨 Apple "What's New" Kuralları

### ✅ İzin verilen
- Yeni özellik duyurusu
- Bug fix listesi
- Performans iyileştirmeleri
- Teşekkür mesajı
- Geri bildirim isteği

### ❌ Yasak
- "Various bug fixes" (Apple bunu yetersiz buluyor) — **spesifik ol**
- Boş veya tek cümlelik update
- Geçen sürümle aynı metin
- Rakip marka adı
- Şarkı sözü, telif sorunlu içerik

### Apple Reviewer Önerisi
- En az 3-5 spesifik değişiklik listele
- Emoji 2-3 taneyle sınırla (overuse spam görünür)
- Friendly, conversational ton kullan

## 📅 Versioning Stratejisi

```
v1.0.0+1   → Launch
v1.0.1+2   → Critical bug fix (1 hafta içinde)
v1.1.0+3   → Bug fixes + small features (3-4 hafta)
v1.2.0+4   → Feature drop (6-8 hafta)
v2.0.0+10  → Major redesign (6 ay)
```

**Semantic versioning kuralı:**
- `MAJOR.MINOR.PATCH+BUILD`
- MAJOR: Breaking change (DB migration vb)
- MINOR: Feature ekleme
- PATCH: Bug fix
- BUILD: Her upload'da artar (App Store Connect zorunlu)
